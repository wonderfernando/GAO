# Gerador Automático de Rotas - Next.js + Prisma + Zod

Este script gera automaticamente **CRUD completo** para todos os modelos do seu `schema.prisma`, incluindo:

- Rotas REST (`GET`, `POST`, `PUT`, `DELETE`)
- Rotas com `[id]` para manipulação de registros individuais
- Sub-rotas recursivas (`api/categorias/[id]/usuarios/route.ts`)
- Validação usando **Zod**
- Filtros inteligentes e paginação (`page` e `limit`)
- Autenticação JWT com **roles**
- Comentários JSDoc opcionais
